    "# Simulated Annealing — Probability Distribution Over Solution Space\n",
    "\n",
    "With SA, the probability of being in state $x$ at temperature $T$ follows the **Boltzmann distribution**:\n",
    "$$P(x) = \\frac{e^{-\\text{cost}(x)\\,/\\,T}}{Z}, \\quad Z = \\sum_{x'} e^{-\\text{cost}(x')\\,/\\,T}$$\n",
    "\n",
    "At **high $T$**: all solutions nearly equally likely (flat distribution).  \n",
    "At **low $T$**: probability concentrates entirely on the optimal solution(s).  \n",
    "\n",
    "We use **7 cities** so we can enumerate *all* $6! = 720$ possible tours exactly and watch probability mass shift in real time."
   ]

    "## 2 · Boltzmann Probability at a Single Temperature\n",
    "Formula: $P(x) \\propto e^{-\\text{cost}(x)/T}$"
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "def boltzmann_probs(costs, T):\n",
    "    \"\"\"Compute normalised Boltzmann probabilities for given temperature T.\"\"\"\n",
    "    log_weights = -costs / T\n",
    "    log_weights -= log_weights.max()   # numerical stability\n",
    "    weights = np.exp(log_weights)\n",
    "    return weights / weights.sum()\n",
    "\n",
    "# Show the formula and a single example\n",
    "T_example = 500\n",
    "probs = boltzmann_probs(costs_sorted, T_example)\n",
    "\n",
    "fig, ax = plt.subplots(figsize=(12, 3.5))\n",
    "ax.bar(range(n_routes), probs, color='#58a6ff', width=1.0, linewidth=0, alpha=0.85)\n",
    "ax.axvline(0, color='#3fb950', linewidth=1.5, linestyle='--', label='Global optimum')\n",
    "ax.set_xlabel('Tours ranked by cost  (best → worst)')\n",
    "ax.set_ylabel('P(tour)')\n",
    "ax.set_title(f'Boltzmann Probability Distribution  at  T = {T_example}', fontsize=12, pad=10)\n",
    "ax.legend(framealpha=0.15, edgecolor='#30363d')\n",
    "ax.grid(True, axis='y', linestyle='--')\n",

    "## 5 · Solution Space Map — 2D Embedding\n",
    "Each dot is one of the 720 tours, projected into 2D by **MDS** on route-swap distances.  \n",
    "Colour = tour cost. Size = Boltzmann probability at snapshot temperature.  \n",
    "The optimal tour glows brightest; nearby tours (structurally similar) also attract mass."
   ]
  },
  {
   "cell_type": "code",
   "execution_count": null,
   "metadata": {},
   "outputs": [],
   "source": [
    "# ── Pairwise route distance: number of edges NOT shared ─────────────────────\n",
    "def edge_set(route):\n",
    "    n = len(route)\n",
    "    return frozenset(\n",
    "        (min(route[i], route[(i+1)%n]), max(route[i], route[(i+1)%n]))\n",
    "        for i in range(n)\n",
    "    )\n",
    "\n",
    "print('Computing pairwise edge distances… (may take ~30 s)')\n",
    "edge_sets = [edge_set(r) for r in all_routes]\n",
    "n         = len(all_routes)\n",
    "D         = np.zeros((n, n))\n",
    "for i in range(n):\n",
    "    for j in range(i+1, n):\n",
    "        shared    = len(edge_sets[i] & edge_sets[j])\n",
    "        d         = N - shared            # N edges total; fewer shared = farther\n",
    "        D[i,j] = D[j,i] = d\n",
    "\n",
    "mds    = MDS(n_components=2, metric=True, dissimilarity='precomputed', random_state=SEED, n_init=1, max_iter=200, normalized_stress=False)\n",
    "coords = mds.fit_transform(D)\n",
    "print('MDS embedding done ✓')"
